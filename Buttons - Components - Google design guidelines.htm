Product Media Carousel

A carousel clearly communicates what action will occur when the user touches it. It consists of text, an image, a video, or a spinning 360 view of the product, and is designed to showcase the best aspects of the current product.

Primary Content

Interactive content in the primary media container should indicate the main action used in the interaction. Examples include tap for full-screen, gestures, and altering mouse icon hover state. The emphasis should always be on the content - as such primary interactivity refers to interactions directly with the content itself, whereas secondary interactivity refers to actions that support the main content.
